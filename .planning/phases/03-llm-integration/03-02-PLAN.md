  <behavior>
    - `LLMValidationError(LLMError)` raised when Pydantic model_validate_json fails inside LLMClient.complete_structured. raw_response attribute holds the bad JSON string.
    - `RetryingLLMClient(LLMClient)` wraps an inner LLMClient. Its `complete_structured` calls inner.complete_structured once; on `LLMValidationError`, retries once with a corrective system message appended (`{"role":"system","content":"Your previous response was not valid JSON matching the schema. Respond ONLY with valid JSON."}`). On second `LLMValidationError`, re-raises.
